# 🏏 IPL Analysis Dashboard in Excel

An interactive IPL Analysis Dashboard created using Microsoft Excel to analyze team performance, toss impact, venue statistics, and player achievements through dynamic charts and visualizations.

---

## 📊 Dashboard Highlights

- Matches Win Analysis (Bat First vs Field First)
- Toss Decision Based Winning %
- Top 10 Venues Analysis
- IPL Winners Distribution Treemap
- Top 10 Player of the Match Award Winners

---

## 🚀 Project Objective

The goal of this project was to transform raw IPL data into meaningful insights using Excel and create an interactive dashboard for better visualization and analysis.
